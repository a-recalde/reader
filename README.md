# Setup

I think `python3` and `tk` should be readily available, if not let me know.

# Run it

```
python3 app.py
```
# Usage
Space Bar to Begin (auto scroll)
Up arrow to speed up
Down arrow to slow down

Manual Scroll
Left to go back
Right to go forward

ESC to exit application
